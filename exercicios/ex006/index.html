Negrito / Destaque

Nesta frase tesmos um termo em negrito usando a tag <B>. (Não Semântica)

Nesta segunda frase o termo em destaque esta usando a tag <STRONG>. (Semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálco usando a tag <I> (Não Semântica)

Nesta segunda frase o termo em itálico esta entre a tag <EM>. (semântica)

Texto Marcado

Nesta frase o termo marcado esta entre a tag <MARK>

Texto Grande texto Pequeno

O texto Grande como esse se usa a tag <big> porém ela esta obsoleta.
O texto pequeno como esse se usa a tag <SMALL> que ainda não esta obsoleta.

Texto Deletado

O Texto deletado como esse aqui se usa a tag <DEL>.

Texto Sublinhado

O texto inserido ou sublinhado como esse se usa a tag <INS>.

Texto sobescrito e subescrito

Um exemplo de sobescrito se usa a tag <SUP>: x20+5
Um exemplo de subescrito se usa a tag <SUB> H2O
